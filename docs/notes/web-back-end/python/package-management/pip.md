# pip

## 简介

Python 官方指定包管理工具

是 Python 自带包无需单独下载，开箱即用

> 一般建议将包下载到[虚拟环境](../index.md#虚拟环境)中，以防止安装的包污染到系统环境

## 常用命令

### pip install \<name>

```shell
pip install <name>
pip install <name>==版本
```

---

### pip install --upgrade \<name>

```shell
pip install --upgrade pip
```

---

### pip install -r requirements.txt

```shell
pip install -r requirements.txt
```
