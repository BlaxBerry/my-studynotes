## 安装

> 本文为 Mac 环境下使用 [asdf](../../dev-tools/asdf/index.md)

```shell
asdf plugin add dart
asdf plugin add flutter
asdf plugin add ruby

asdf install dart 3.0.5
asdf install flutter 3.10.5-stable
asdf install ruby 3.2.2
```
