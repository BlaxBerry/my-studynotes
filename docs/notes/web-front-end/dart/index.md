# Dart 相关

![](/images/dart.webp)

## 简介

Dart 是谷歌开发的类型安全、面向对象的语言，与 JavaScript 相似度较高

主要应用于 Web、服务器、移动应用、物联网等领域（ 尤其是 Flutter ）

## 安装

> 本文为 Mac 环境下使用 [asdf](../../dev-tools/asdf/index.md)

一般安装 [Flutter](../cross-device/flutter/index.md) 时顺带安装

## 代码执行

```shell
dart 文件.dart
```
