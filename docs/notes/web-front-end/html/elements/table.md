### \<table>

---

### \<thead>、\<tbody>、\<tfoot>

```html
<table>
  <thead>
    <!-- 表头 -->
  </thead>
  <tbody>
    <!-- 表体 -->
  </tbody>
  <tfoot>
    <!-- 表尾 -->
  </tfoot>
</table>
```

### \<tr>、\<th>、\<td>

```html
<table>
  <tr>
    <th>标题1</th>
    <th>标题2</th>
    <th>标题3</th>
  </tr>
  <tr>
    <td>单元格1</td>
    <td>单元格2</td>
    <td></td>
  </tr>
  <tr>
    <td>单元格4</td>
    <td></td>
    <td>单元格6</td>
  </tr>
  <tr>
    <td></td>
    <td>单元格8</td>
    <td>单元格9</td>
  </tr>
</table>
```
