## 运行环境

JavaScript 有两个运行环境：

::: details 1. 在浏览器中执行

只需要写入在 HTML 中的`<script>`标签中

```html
<html>
  <header>
    <script src="路径"></script>
  </header>
  <body>
    <script>
      // JS 逻辑
    </script>
  </body>
</html>
```

:::

::: details 2. 在终端中执行

详见 [Node.js](../../web-back-end/node-js/index.md#安装)

:::
