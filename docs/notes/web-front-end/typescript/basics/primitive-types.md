# TS 原始类型 ( Primitive Types )

## boolean

```ts
let data: boolean;

data = true;
data = false;
```

## number

```ts
let data: number;

data = Infinity;
data = -Infinity;
data = NaN;
data = -0;
data = 0;
data = 0.1;
```

## string

```ts
let data: string;

data = "xxx";
data = `xxx`;
```
