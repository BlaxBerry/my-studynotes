# React 钩子函数 ( Hooks )

## 简介

Hook 是 React 16.8 的新增特性

可以让在不编写 class 的情况下使用 state 以及其他的 React 特性

- 解决 hoc 的 prop 重名问题；
- 解决 render props 因共享数据 而出现嵌套地狱的问题；
- 能在 return 之外使用数据
- hook 只能在组件顶层使用，
